## 🎨 رنگ‌ها و معنی آن‌ها

### امتیازات
- 🟢 **سبز (8-10)**: عالی، عملکرد فوق‌العاده
- 🔵 **آبی (6-8)**: خوب، عملکرد مناسب
- 🟠 **نارنجی (4-6)**: متوسط، نیاز به بهبود
- 🔴 **قرمز (1-4)**: ضعیف، نیاز به توجه فوری

### وضعیت
- ✅ **سبز**: تکمیل شده، فعال، موفق
- ⏰ **نارنجی**: در انتظار، نیاز به اقدام
- ⛔ **خاکستری**: غیرفعال، متوقف شده

### نقش‌ها
- 🔴 **قرمز**: Admin
- 🟣 **بنفش**: معاون فنی
- 🔵 **آبی**: استراتژیست
- 🟣 **بنفش**: نویسنده

## ⚡ نکات سریع

### چگونه به سرعت عملکرد سیستم را ببینم؟
```
1. نگاه به کارت‌های بالا
2. بررسی میانگین امتیازات (باید بالای 7 باشد)
3. چک کردن نرخ تکمیل (باید بالای 80% باشد)
```

### چگونه بهترین‌ها را پیدا کنم؟
```
تب "نمای کلی" > بخش "برترین استراتژیست‌ها/نویسندگان"
```

### چگونه روند زمانی را ببینم؟
```
تب "روند عملکرد" > نمودار خطی
```

### چگونه کارگروه‌ها را مقایسه کنم؟
```
تب "کارگروه‌ها" > نمودار میله‌ای و کارت‌های جزئیات
```

### چگونه آخرین رویدادها را ببینم؟
```
تب "فعالیت‌ها" > لیست ارزیابی‌های اخیر و تایم‌لاین
```

## 💡 سناریوهای کاربردی

### سناریو 1: بررسی صبحگاهی (2 دقیقه)
```
1. ورود به داشبورد
2. نگاه سریع به کارت‌های بالا
3. چک کردن تب "فعالیت‌ها" برای رویدادهای دیروز
```

### سناریو 2: گزارش هفتگی (5 دقیقه)
```
1. تب "نمای کلی" - بررسی برترین‌ها
2. تب "روند عملکرد" - بررسی نمودار خطی
3. تب "کارگروه‌ها" - شناسایی کارگروه‌های ضعیف
```

### سناریو 3: گزارش ماهانه (10 دقیقه)
```
1. بررسی همه کارت‌های آمار
2. تحلیل نمودارهای روند
3. مقایسه کارگروه‌ها
4. بررسی لیست برترین‌ها
5. تصمیم‌گیری برای ماه آینده
```
